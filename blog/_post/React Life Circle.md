组件的生命周期可分成三个状态：

* Mounting：已插入真实 DOM

- Updating：正在被重新渲染
- Unmounting：已移出真实 DOM

![](/images/react-life-cycle-new.jpeg)



## React vs Vue

Vue本质是MVVM框架，由MVC发展而来；

React是前端组件化框架，由后端组件化发展而来。
